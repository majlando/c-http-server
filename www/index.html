c-http-server

A tiny educational non-blocking HTTP server in C (epoll + small parser)

Quickstart

Build, run, and request the homepage:

# build
make

# run (listens on 0.0.0.0:8080)
./bin/c-http-server

# from another terminal
curl -i http://127.0.0.1:8080/

Features

  • Non-blocking I/O with epoll (Linux)
  • Incremental request parsing (handles fragmented input)
  • Static file serving from /www
  • Small unit tests (see /tests)

Notes

  • No TLS — use a reverse proxy for production.
  • No chunked transfer-encoding support yet.
  • Tests and ASan target available: make test, make asan.

Contribute

Small PRs welcome. Good first changes:

  1. Map more MIME types
  2. Return 413 on oversized requests
  3. Add timeouts and connection caps
