c-http-server

Minimal educational HTTP server in C — epoll, non-blocking I/O, and a small incremental parser

Overview

This repository is a compact, readable starting point for learning low-level HTTP server internals. The server is intentionally small and focuses on clarity rather than completeness.

  • Non-blocking sockets using epoll (Linux)
  • Incremental HTTP request parsing (handles fragmented input)
  • Static file serving from the /www directory
  • Small test-suite for the parser

Quick demo

Build and run locally, then request the homepage:

# build
make

# run (listens on port 8080)
./bin/c-http-server

# from another terminal
curl -i http://127.0.0.1:8080/

Notes & limitations

This project is an educational scaffold — it lacks full HTTP compliance. Notable limitations:

  • No TLS support
  • Limited header/body handling (no chunked encoding support yet)
  • Basic MIME mapping only
  • Simple logging to server.log

Repository

Static files live in /www. Open /www/index.html to edit this page.
